# ssafy-finance-project

소비 패턴을 분석하고 사용자의 잠재 저축액을 계산한 뒤  
금융 상품 정보를 활용해 예금 추천까지 제공하는 금융 추천 서비스 프로젝트입니다.

---

## 1. 프로젝트 개요

- **프로젝트명**  
  ssafy finance project

- **목표**  
  일상 속 사소한 소비를 기록하고  
  "이 소비를 줄였으면 얼마를 모을 수 있었을까"를 계산하여  
  사용자가 자신의 소비 습관을 인식하고 더 나은 금융 의사결정을 할 수 있도록 돕는 서비스입니다.

- **핵심 컨셉**  
  - 불필요 소비를 줄여서  
  - 잠재 저축액을 계산하고  
  - 금융감독원 오픈 API로 가져온 예금 상품에 넣었을 때  
  - 얼마를 모을 수 있는지 시뮬레이션과 함께 **금융 상품을 추천**합니다.

---

## 2. 주요 기능

1. **소비 기록 관리**
   - 날짜, 항목명, 카테고리, 금액, 불필요 소비 여부를 입력
   - 소비 내역에 대한 조회, 수정, 삭제 기능 제공

2. **잠재 저축액 계산**
   - 이번 달 불필요 소비 합계를 계산
   - 현재 패턴이 1년간 유지된다고 가정했을 때의 예상 저축액 계산

3. **금융 상품 데이터 연동**
   - 금융감독원 금융상품 통합 비교 공시 API를 활용하여 예금 상품 데이터 수집
   - 은행, 상품명, 금리, 가입 기간 등의 정보를 내부 DB에 저장

4. **예금 추천 및 시뮬레이션**
   - 사용자의 잠재 저축액과 투자 성향 정보를 바탕으로 예금 상품 추천
   - 추천 상품에 해당 금액을 예치했을 때 1년 후 예상 금액 계산

5. **대시보드 및 리포트**
   - 이번 달 총 지출과 불필요 소비 비율 요약
   - 카테고리별 소비 패턴 분석
   - 추천 예금 상품을 카드 형식으로 제공
